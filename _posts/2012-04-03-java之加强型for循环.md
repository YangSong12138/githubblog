什么是加强型for循环
-------------------
类似下面的代码：

	for(int a : Array){
		pass;
	}

实现的原理
----------

其实就是在底层封装了一层基本for循环。

使用环境
--------

1.对元素进行顺序访问；

2.数组或集合中的元素访问；

3.不需要指定访问某个元素或提供当前索引。


注意事项
--------

访问数组或者iterable类型变量(实现iterable接口)，使用它比较快捷。
